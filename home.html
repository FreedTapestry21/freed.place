Hosting

This section is for those wondering how I'm hosting this website.
I have 2 home servers, which one of them is running Proxmox.
Proxmox is an operating system used for virtualization.
For example, thanks to Proxmox I have around 7 virtualized server doing all sorts of things!
One of those things being hosting my website.
I also use Cloudflare for reverse proxy and DDOS protection.
